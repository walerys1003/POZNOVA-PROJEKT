# POZNOVA-PROJEKT

  Instytucjonalna strona internetowa POSNOVA — polskiej spółdzielni socjalnej nowej generacji łączącej technologię AI z misją społeczną.

  ## O projekcie

  **POSNOVA** to pełny mockup strony instytucjonalnej zbudowany w React + TypeScript z:

  - 🎨 **Animacje Canvas 2D** — CrystalHero (strona główna), NeuralHero (Lab), ParticleHumans (Wpływ)
  - 🔀 **React Router v7** — pełna nawigacja URL-based (HashRouter)
  - 🇵🇱 **9 stron po polsku** — Landing, Usługi, Produkty, Lab, Wpływ, Instytucja, Kariera, Wiedza, Kontakt
  - ⚫ **Ścisły design system** — czerń / biel / odcienie szarości + akcent #00A3B4
  - ✍️ **Typografia** — Playfair Display + Inter + JetBrains Mono

  ## Strony

  | Ścieżka | Opis |
  |---------|------|
  | `/` | Landing — hero z animowanym kryształem 3D |
  | `/uslugi` | Oferta usług B2B |
  | `/produkty` | Portfolio 6 produktów SaaS (PZN LABS) |
  | `/lab` | Instytut badawczy — AI, LegalTech, Social Impact |
  | `/wplyw` | Mierzony wpływ społeczny z animacją cząsteczkowych postaci |
  | `/instytucja` | Manifest, architektura, zespół, status prawny |
  | `/kariera` | POSNOVA Academy i oferty pracy |
  | `/wiedza` | Blog i baza wiedzy |
  | `/kontakt` | Formularze: Biznes, Inwestorzy, Media, Granty, Partnerzy |

  ## Technologie

  - React 18 + TypeScript
  - Vite 5
  - react-router-dom v7 (HashRouter)
  - Canvas 2D API (animacje hero)
  - JetBrains Mono, Playfair Display, Inter

  ## Uruchomienie

  ```bash
  pnpm install
  pnpm --filter @workspace/mockup-sandbox run dev
  ```
